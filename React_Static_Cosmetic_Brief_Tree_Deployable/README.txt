# Cosmetic Brief Tree Static Demo

This package contains:
- `public/index.html`: Static demo of your cosmetic decision tree
- `Cosmetic_Brief_Tree_Static.pdf`: Printable version of key insights
- `package.json`: Config to host or expand this as a React app

## How to Use

To preview locally:
1. Unzip the contents
2. Run `npx serve public` or drag `index.html` into a browser

To deploy:
- Upload to Netlify or Vercel using React preset (React Scripts)
